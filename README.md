# EasyRenderer
EasyRenderer is a quick but highly inefficient solution for graphics.

### Advantages
- Easy to setup
- Easy to work with
- Easy to understand
- No external libraries required
- Has many fancy and useless functionalities

### Disadvantages
- Poor performance
- Single Threaded
- The autor (me) is braindead
